Graph of Vk friends API

Parameters:

  • user_id(int)[required]: User ID in the social network Vk.
  • email(str)[required]: Email where we will send the result. Result it's a high resolution photo and a brief description.
  • accuracy(float)[optional]: Part of the friends of each friend of the main user. Must be in range [0,1].

Methods: GET

Description: The API analyzes the primary user's friends and each friend's friends and creates a graph.

Examples:

  • http://127.0.0.1:8000/graph/user_id=123456789&email=your_email@gmail.com
  • http://127.0.0.1:8000/graph/user_id=123456789&email=your_email@gmail.com?accuracy=0.2
